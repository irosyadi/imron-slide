3.20

That’s an excellent problem to solidify your understanding of RLC circuits and their representation as Linear Time-Invariant (LTI) systems. This parallel RLC topology is often used in filtering applications, which is fundamental in Electrical and Computer Engineering. Let’s tackle this step-by-step.

(a) Find the differential equation relating \(x(t)\) and \(y(t)\)

To find the differential equation, we’ll use Kirchhoff’s Voltage Law (KVL) and the constitutive relationships for the components.

  1. Component Relationships:

    • The voltage across the capacitor, \(y(t)\), is also the voltage across the inductor and the resistor/source combination.
    • The current through the capacitor is \(i_C(t) = C \frac{dy(t)}{dt}\).
    • The current through the inductor is \(i_L(t)\).
    • The voltage across the inductor is \(v_L(t) = L \frac{di_L(t)}{dt}\). Since it’s in parallel with the capacitor, \(v_L(t) = y(t)\).

  2. Relate Inductor Current and Output Voltage: From \(v_L(t) = y(t)\) and \(v_L(t) = L \frac{di_L(t)}{dt}\), we can find \(i_L(t)\) by integrating: \[i_L(t) = \frac{1}{L} \int y(t) dt\]

  3. Apply KVL to the Input Loop: The voltage source \(x(t)\) equals the voltage drop across the resistor (\(v_R(t)\)) plus the voltage drop across the parallel LC branch (\(y(t)\)). \[x(t) = v_R(t) + y(t)\]

  4. Express \(v_R(t)\) in terms of \(y(t)\): The current through the resistor, \(i_R(t)\), is the total current leaving the source, which splits into \(i_L(t)\) and \(i_C(t)\). \[i_R(t) = i_L(t) + i_C(t)\] Using the component relationships from step 1 and 2: \[i_R(t) = \left(\frac{1}{L} \int y(t) dt\right) + \left(C \frac{dy(t)}{dt}\right)\] The resistor voltage is \(v_R(t) = R \cdot i_R(t)\): \[v_R(t) = R \left[ \frac{1}{L} \int y(t) dt + C \frac{dy(t)}{dt} \right]\]

  5. Substitute into KVL and Differentiate: Substitute \(v_R(t)\) back into the KVL equation: \[x(t) = R \left[ \frac{1}{L} \int y(t) dt + C \frac{dy(t)}{dt} \right] + y(t)\] To eliminate the integral, differentiate the entire equation with respect to \(t\): \[\frac{dx(t)}{dt} = R \left[ \frac{1}{L} y(t) + C \frac{d^2 y(t)}{dt^2} \right] + \frac{dy(t)}{dt}\]

  6. Final Differential Equation (with R=1, L=1, C=1): Substitute the given values \(R=1 \ \Omega\), \(L=1 \ H\), \(C=1 \ F\): \[\frac{dx(t)}{dt} = 1 \left[ \frac{1}{1} y(t) + 1 \frac{d^2 y(t)}{dt^2} \right] + \frac{dy(t)}{dt}\] Rearranging into standard form: \[\frac{d^2 y(t)}{dt^2} + \frac{dy(t)}{dt} + y(t) = \frac{dx(t)}{dt}\]

(b) Determine the frequency response \(H(j\omega)\)

The frequency response \(H(j\omega)\) is the ratio of the steady-state output phasor \(Y(j\omega)\) to the input phasor \(X(j\omega)\) when the input is \(x(t)=e^{j\omega t}\).

The easiest way to find this is to use the Phasor (or Impedance) approach on the circuit.

  1. Transform Components to Impedances:

    • Resistor \(R \rightarrow Z_R = R\)
    • Inductor \(L \rightarrow Z_L = j\omega L\)
    • Capacitor \(C \rightarrow Z_C = \frac{1}{j\omega C}\)

  2. Analyze the Circuit in the Frequency Domain: The circuit is a voltage divider. The output voltage \(Y(j\omega)\) is across the parallel combination of \(L\) and \(C\), \(Z_{LC}\). \[Z_{LC} = Z_L || Z_C = \frac{Z_L Z_C}{Z_L + Z_C} = \frac{(j\omega L) \left(\frac{1}{j\omega C}\right)}{j\omega L + \frac{1}{j\omega C}} = \frac{L/C}{j\omega L + \frac{1}{j\omega C}}\]

  3. Apply Voltage Divider Rule: \[Y(j\omega) = X(j\omega) \left[ \frac{Z_{LC}}{Z_R + Z_{LC}} \right]\] The frequency response is \(H(j\omega) = \frac{Y(j\omega)}{X(j\omega)}\): \[H(j\omega) = \frac{Z_{LC}}{R + Z_{LC}} = \frac{1}{1 + R/Z_{LC}}\]

  4. Substitute \(Z_{LC}\) and Component Values (R=1, L=1, C=1): \[Z_{LC} = \frac{1/1}{j\omega(1) + \frac{1}{j\omega(1)}} = \frac{1}{j\omega + \frac{1}{j\omega}} = \frac{1}{\frac{-\omega^2+1}{j\omega}} = \frac{j\omega}{1-\omega^2}\]

    Now, substitute \(Z_{LC}\) and \(R=1\) into \(H(j\omega)\): \[H(j\omega) = \frac{Z_{LC}}{1 + Z_{LC}} = \frac{\frac{j\omega}{1-\omega^2}}{1 + \frac{j\omega}{1-\omega^2}} = \frac{\frac{j\omega}{1-\omega^2}}{\frac{1-\omega^2 + j\omega}{1-\omega^2}}\] Simplifying the expression: \[H(j\omega) = \frac{j\omega}{1-\omega^2 + j\omega}\]

(c) Determine the output \(y(t)\) if \(x(t)=\sin (t)\)

For an LTI system, if the input is a sinusoid \(x(t)=A \cos(\omega_0 t + \phi)\), the steady-state output \(y(t)\) will also be a sinusoid at the same frequency \(\omega_0\), with a scaled magnitude and a shifted phase: \[y(t) = A |H(j\omega_0)| \cos(\omega_0 t + \phi + \angle H(j\omega_0))\]

  1. Identify Input Parameters: The input is \(x(t) = \sin(t)\). We can write this as \(\cos(t - \pi/2)\). The frequency is \(\omega_0 = 1 \ \text{rad/s}\).

  2. Evaluate \(H(j\omega)\) at \(\omega = \omega_0 = 1\): Substitute \(\omega=1\) into the frequency response found in part (b): \[H(j\omega) = \frac{j\omega}{1-\omega^2 + j\omega}\] \[H(j1) = \frac{j(1)}{1-(1)^2 + j(1)} = \frac{j}{1-1+j} = \frac{j}{j} = 1\]

  3. Determine Magnitude and Phase: The magnitude is \(|H(j1)| = |1| = 1\). The phase is \(\angle H(j1) = \angle(1) = 0\) radians.

  4. Calculate the Output \(y(t)\): The output is \(y(t) = |H(j1)| \sin(\omega_0 t + \angle H(j1))\). \[y(t) = 1 \cdot \sin(1 \cdot t + 0)\] \[\boldsymbol{y(t) = \sin(t)}\]

Interpretation (Mentorship Note): Notice that at \(\omega = 1 \ \text{rad/s}\), which is the resonant frequency (\(\omega_0 = 1/\sqrt{LC} = 1/\sqrt{1\cdot1} = 1\)), the parallel LC tank acts as an open circuit (infinite impedance) if the resistor \(R\) were absent. Because \(R\) is present, the effective impedance of the parallel LC combination is exactly \(Z_{LC} = j\omega/(1-\omega^2)\). At resonance (\(\omega=1\)), \(Z_{LC}\) becomes singular (\(j/0\)), but when \(R\) is included, the full response \(H(j\omega)\) is well-behaved, resulting in a simple gain of 1. The output voltage is the same as the input voltage, meaning the \(R\) branch is effectively seeing a zero-impedance path in the frequency domain at resonance.

Problem 4.20

That’s the logical next step in our analysis! Since we have the frequency response \(H(j\omega)\) from the previous problem, finding the impulse response \(h(t)\) requires the inverse Fourier Transform.

The system’s frequency response for \(R=1, L=1, C=1\) was determined to be: \[H(j\omega) = \frac{j\omega}{1-\omega^2 + j\omega}\]

1. Relate \(H(j\omega)\) to the Transfer Function \(H(s)\)

For LTI systems, it’s often easier to find the impulse response using the Laplace Transform (which gives us the transfer function \(H(s)\)) and then taking the inverse Laplace Transform. Since \(H(j\omega) = H(s)|_{s=j\omega}\), we can find \(H(s)\) by replacing \(j\omega\) with \(s\):

\[H(s) = \frac{s}{1-s^2 + s} = \frac{s}{s^2 + s + 1}\]

2. Find the Poles of \(H(s)\)

To perform the inverse Laplace Transform, we first need to find the roots of the denominator (the characteristic equation) to identify the system’s poles. \[s^2 + s + 1 = 0\]

Using the quadratic formula, \(s = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\): \[s = \frac{-1 \pm \sqrt{1^2 - 4(1)(1)}}{2} = \frac{-1 \pm \sqrt{-3}}{2} = \frac{-1}{2} \pm j\frac{\sqrt{3}}{2}\]

The poles are a complex conjugate pair, indicating the system is underdamped: \[p_1 = -\frac{1}{2} + j\frac{\sqrt{3}}{2}\] \[p_2 = -\frac{1}{2} - j\frac{\sqrt{3}}{2}\]

3. Perform Partial Fraction Expansion

We can now express \(H(s)\) using its partial fraction expansion: \[H(s) = \frac{s}{(s - p_1)(s - p_2)} = \frac{A_1}{s - p_1} + \frac{A_2}{s - p_2}\]

Where \(A_1\) and \(A_2\) are the residues. Since the coefficients of the transfer function are real, \(A_2 = A_1^*\).

Calculate \(A_1\): \[A_1 = \lim_{s \to p_1} (s - p_1) H(s) = \left. \frac{s}{s - p_2} \right|_{s = p_1}\] \[A_1 = \frac{p_1}{p_1 - p_2}\]

Substitute the pole values: \[p_1 - p_2 = \left(-\frac{1}{2} + j\frac{\sqrt{3}}{2}\right) - \left(-\frac{1}{2} - j\frac{\sqrt{3}}{2}\right) = j\sqrt{3}\]

\[A_1 = \frac{-\frac{1}{2} + j\frac{\sqrt{3}}{2}}{j\sqrt{3}} = \frac{1}{j\sqrt{3}} \left(-\frac{1}{2} + j\frac{\sqrt{3}}{2}\right) = \frac{1}{j\sqrt{3}} \left(-\frac{1}{2}\right) + \frac{1}{j\sqrt{3}} \left(j\frac{\sqrt{3}}{2}\right)\] \[A_1 = \frac{1}{2} - \frac{1}{j2\sqrt{3}} = \frac{1}{2} + j\frac{1}{2\sqrt{3}}\]

4. Inverse Laplace Transform to find \(h(t)\)

Since the system is causal, the region of convergence (ROC) is \(\text{Re}\{s\} > -1/2\), which contains the \(j\omega\)-axis.

The general inverse transform for a complex conjugate pair of poles is: \[h(t) = 2 \text{Re}\left\{ A_1 e^{p_1 t} \right\} u(t)\]

First, express \(A_1\) and \(p_1\) in polar form. Pole \(p_1\): \[\sigma = -\frac{1}{2}, \quad \omega_d = \frac{\sqrt{3}}{2}\] \[p_1 = \sigma + j\omega_d = e^{-\sigma t} e^{j\omega_d t} = e^{-t/2} e^{j\sqrt{3}t/2}\]

Residue \(A_1\): \[|A_1| = \sqrt{\left(\frac{1}{2}\right)^2 + \left(\frac{1}{2\sqrt{3}}\right)^2} = \sqrt{\frac{1}{4} + \frac{1}{12}} = \sqrt{\frac{3+1}{12}} = \sqrt{\frac{4}{12}} = \sqrt{\frac{1}{3}} = \frac{1}{\sqrt{3}}\] \[\angle A_1 = \arctan\left(\frac{1/(2\sqrt{3})}{1/2}\right) = \arctan\left(\frac{1}{\sqrt{3}}\right) = \frac{\pi}{6}\]

\(A_1 = \frac{1}{\sqrt{3}} e^{j\pi/6}\)

Now, substitute into the time-domain expression: \[h(t) = 2 \text{Re}\left\{ \left(\frac{1}{\sqrt{3}} e^{j\pi/6}\right) \left(e^{-t/2} e^{j\sqrt{3}t/2}\right) \right\} u(t)\] \[h(t) = \frac{2}{\sqrt{3}} e^{-t/2} \text{Re}\left\{ e^{j(\sqrt{3}t/2 + \pi/6)} \right\} u(t)\]

Using Euler’s identity \(\text{Re}\{e^{j\theta}\} = \cos(\theta)\): \[h(t) = \frac{2}{\sqrt{3}} e^{-t/2} \cos\left(\frac{\sqrt{3}}{2}t + \frac{\pi}{6}\right) u(t)\]

The impulse response of the causal LTI RLC system is: \[\boldsymbol{h(t) = \frac{2}{\sqrt{3}} e^{-t/2} \cos\left(\frac{\sqrt{3}}{2}t + \frac{\pi}{6}\right) u(t)}\] This is the characteristic response of a second-order underdamped system, which makes perfect sense for an RLC circuit.

Problem 9.18

This is a fantastic set of questions that connects the transfer function in the \(s\)-plane to the practical behavior of the circuit in the frequency domain. We’ll use our previous results for the case \(R=1, L=1, C=1\) and then analyze the effect of a major change in the resistance.

(a) Determine \(H(s)\) and specify its region of convergence (ROC) for \(R=1\)

The transfer function \(H(s)\) was derived from the frequency response \(H(j\omega)\) by replacing \(j\omega\) with \(s\).

\[H(s) = \frac{s}{s^2 + s + 1}\]

Poles and ROC

To specify the Region of Convergence (ROC), we need the poles of the system, which are the roots of the denominator: \(s^2 + s + 1 = 0\).

The poles are: \[p_{1, 2} = -\frac{1}{2} \pm j\frac{\sqrt{3}}{2}\]

The real part of the poles is \(\sigma = -\frac{1}{2}\).

  1. Causality Requirement: Since the system is specified as causal, the ROC must be a right-half plane to the right of the rightmost pole.
  2. Stability Requirement: Since the system is also stable (for an LTI system to be stable, the ROC must include the \(j\omega\)-axis), the real part of all poles must be negative. This is true since \(\text{Re}\{p_{1, 2}\} = -1/2 < 0\).

Therefore, the Region of Convergence (ROC) is: \[\text{ROC}: \text{Re}\{s\} > -\frac{1}{2}\]

(b) Geometric Evaluation of \(|H(j\omega)|\) for \(R=1\) (Bandpass Characteristic)

The frequency response magnitude \(|H(j\omega)|\) is found by geometrically evaluating the ratio of vector magnitudes from the pole-zero plot to the \(j\omega\)-axis.

Pole-Zero Plot (\(R=1\))

  • Zero: \(z_1 = 0\)

  • Poles: \(p_{1, 2} = -\frac{1}{2} \pm j\frac{\sqrt{3}}{2}\)

  • \(|H(j\omega)| = \frac{\text{Magnitude of vector from zero to } j\omega}{\text{Magnitude of vectors from poles to } j\omega}\)

  • \(|H(j\omega)| = \frac{|j\omega|}{|j\omega - p_1| |j\omega - p_2|}\)

Geometric Analysis:

  1. Near \(\omega = 0\) (DC):
    • The vector from the zero at \(s=0\) to \(j\omega\) is very small (length \(\approx 0\)).
    • Therefore, \(|H(j0)| \approx 0\). This attenuates low frequencies. (This is a characteristic of highpass/bandpass filters).
  2. As \(\omega \to \infty\) (High Frequencies):
    • \(|j\omega| \to \infty\)
    • \(|j\omega - p_1| |j\omega - p_2| \approx (j\omega)(j\omega) = -\omega^2\).
    • \(|H(j\omega)| \approx \frac{\omega}{\omega^2} = \frac{1}{\omega} \to 0\). This attenuates high frequencies.
  3. Near the Natural Frequency:
    • The poles are located close to the natural frequency \(\omega_n = \sqrt{1} = 1\) (since \(|p_1| = \sqrt{(-1/2)^2 + (\sqrt{3}/2)^2} = 1\)).
    • When \(\omega\) is close to the magnitude of the poles, the vector from the pole \(p_1\) to \(j\omega\) becomes the shortest, leading to a peak in the magnitude. The peak occurs close to the imaginary part of the poles, \(\omega_d = \sqrt{3}/2 \approx 0.866 \ \text{rad/s}\).

Conclusion for (b): Since \(|H(j\omega)|\) is small at \(\omega=0\) and \(\omega \to \infty\), but peaks at a central frequency, the characteristic is Bandpass.

(c) Determine \(H(s)\) and ROC for \(R = 10^{-3} \Omega\)

The general transfer function (before plugging in values) is: \[H(s) = \frac{L/C}{R\left(\frac{L}{C}\right) + s L + \frac{1}{s C}} = \frac{s/RC}{s + \frac{1}{R} \left(\frac{1}{sL} + sC\right)}\] (Using the impedance method from the previous problem and substituting \(s\) for \(j\omega\) is often more robust.)

Starting from the general impedance form: \[H(s) = \frac{Z_{LC}}{R + Z_{LC}} \quad \text{where } Z_{LC} = \frac{sL}{s^2 LC + 1}\]

With \(L=1, C=1\), the general form is: \[H(s) = \frac{\frac{s}{s^2 + 1}}{R + \frac{s}{s^2 + 1}} = \frac{s}{R(s^2+1) + s} = \frac{s}{R s^2 + s + R}\]

Now, substitute the new value \(R = 10^{-3}\): \[H(s) = \frac{s}{10^{-3} s^2 + s + 10^{-3}}\]

Poles and ROC

To find the poles, we solve \(10^{-3} s^2 + s + 10^{-3} = 0\): \[s = \frac{-1 \pm \sqrt{1^2 - 4(10^{-3})(10^{-3})}}{2(10^{-3})} = \frac{-1 \pm \sqrt{1 - 4 \cdot 10^{-6}}}{2 \cdot 10^{-3}}\]

Since \(4 \cdot 10^{-6}\) is extremely small compared to 1, the square root is approximately \(\sqrt{1} = 1\). This means the poles are two distinct, real-valued poles (the system is overdamped).

Using the approximation \(\sqrt{1 - \epsilon} \approx 1 - \frac{\epsilon}{2}\) with \(\epsilon = 4 \cdot 10^{-6}\): \[\sqrt{1 - 4 \cdot 10^{-6}} \approx 1 - 2 \cdot 10^{-6}\]

Pole 1 (using the ‘+’ sign - the pole closer to the \(j\omega\)-axis): \[p_1 \approx \frac{-1 + (1 - 2 \cdot 10^{-6})}{2 \cdot 10^{-3}} = \frac{-2 \cdot 10^{-6}}{2 \cdot 10^{-3}} = -10^{-3}\]

Pole 2 (using the ‘-’ sign): \[p_2 \approx \frac{-1 - 1}{2 \cdot 10^{-3}} = \frac{-2}{2 \cdot 10^{-3}} = -10^{3}\]

The poles are \(p_1 \approx -0.001\) and \(p_2 \approx -1000\).

The rightmost pole determines the ROC. For a causal and stable system: \[\text{ROC}: \text{Re}\{s\} > -10^{-3}\]

(d) Geometric Evaluation of \(|H(j\omega)|\) for \(R = 10^{-3}\) (Bandpass Characteristic)

Pole-Zero Plot (\(R=10^{-3}\))

  • Zero: \(z_1 = 0\)
  • Poles: \(p_1 \approx -0.001\), \(p_2 \approx -1000\)

Geometric Analysis:

\[|H(j\omega)| = \frac{|j\omega|}{|j\omega - p_1| |j\omega - p_2|}\]

  1. Near \(\omega = 0\) (DC):
    • The vector from the zero at \(s=0\) to \(j\omega\) is small, so \(|H(j0)| \approx 0\). (Lowpass attenuation).
  2. Region \(\omega \in (p_1, p_2)\), e.g., \(\omega = 1\):
    • The distance to the remote pole \(p_2 \approx -1000\) is \(|j\omega - p_2| \approx 1000\) (large, constant).
    • The distance to the nearby pole \(p_1 \approx -0.001\) is \(|j\omega - p_1| \approx \omega\) (or close to 1).
    • The numerator is \(|j\omega| = \omega\).
    • \(|H(j\omega)| \approx \frac{\omega}{(\omega)(1000)} = \frac{1}{1000}\). This is a small but constant gain across the mid-range.
  3. Near \(\omega = 1\) (Resonance, \(\omega_n=1\)):
    • The magnitude will peak where the numerator \(|j\omega|\) is large but the denominator is small.
    • At the resonant frequency \(\omega = 1/\sqrt{LC} = 1\): \[|H(j1)| = \frac{1}{|j1 - (-0.001)| |j1 - (-1000)|} \approx \frac{1}{|1+0.001| |1+1000|} \approx \frac{1}{1 \cdot 1000} = 10^{-3}\]
  4. As \(\omega \to \infty\) (High Frequencies):
    • \(|H(j\omega)| \approx \frac{\omega}{\omega^2} = \frac{1}{\omega} \to 0\). (Highpass attenuation).

Conclusion for (d): The system still has a zero at DC and two finite poles, leading to attenuation at both low and very high frequencies. The overall characteristic remains Bandpass, but the peak is much lower (\(|H(j1)| \approx 10^{-3}\)) and the resonance is much less sharp compared to the \(R=1\) case because \(R\) dominates the response.
