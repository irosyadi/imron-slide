A system is said to be memoryless if its output depends only on what?
A. The input at past times.
B. The input at future times.
C. The input at that same time.
D. The sum of all past inputs.
ANSWER: C
Which of the following is defined as a memoryless system?
A. An accumulator.
B. A capacitor with input as current and output as voltage.
C. A resistor with voltage as output and current as input.
D. A delay element.
ANSWER: C
A system described by \(y[n]=x[n-1]\) is an example of what type of system?
A. Memoryless
B. Invertible
C. With memory
D. Causal only
ANSWER: C
If a system is invertible, what does its inverse system, when cascaded with the original system, yield as an output?
A. A scaled version of the input to the original system.
B. The inverse of the input to the original system.
C. An output equal to the input to the original system.
D. A delayed version of the input to the original system.
ANSWER: C
What is another term used to describe a causal system?
A. Anticipative
B. Unstable
C. Nonanticipative
D. Time-varying
ANSWER: C
If two inputs to a causal system are identical up to some point in time \(t_0\) or \(n_0\), what can be said about their corresponding outputs?
A. They must be different after \(t_0\).
B. They must also be equal up to this same time.
C. They can be arbitrarily different.
D. The output is zero after \(t_0\).
ANSWER: B
Which of the following systems is NOT causal?
A. \( y[n] = x[n] - x[n-1] \)
B. \( y(t) = \frac{1}{C}\int_{-\infty}^{t} x(\tau) d\tau \)
C. \( y[n] = x[n] - x[n+1] \)
D. \( y(t) = x(t) \)
ANSWER: C
According to the formal definition, if the input to a stable system is bounded, what must be true about the output?
A. The output must grow without bound.
B. The output must also be bounded.
C. The output will be zero.
D. The output will be stable if and only if the input is stable.
ANSWER: B
Which of the following is an example of an unstable system?
A. An RC circuit with positive component values.
B. A pendulum in its normal upright position.
C. A bank account balance model with compounding interest on deposits.
D. An automobile with frictional resistance.
ANSWER: C
What property describes a system whose behavior and characteristics are fixed over time, such that a time shift in the input signal results in an identical time shift in the output signal?
A. Causality
B. Linearity
C. Memoryless
D. Time invariance
ANSWER: D
If \(y(t)\) is the output of a continuous-time, time-invariant system when \(x(t)\) is the input, what would be the output when \(x(t-t_0)\) is applied?
A. \(y(t+t_0)\)
B. \(y(t-t_0)\)
C. \(y(t)\)
D. \(y(-t)\)
ANSWER: B
Which of the following discrete-time systems is an example of a time-varying system?
A. \( y[n] = x[n-1] \)
B. \( y[n] = n x[n] \)
C. \( y[n] = u[n] \)
D. \( y[n] = \sum_{k=-\infty}^{n} x[k] \)
ANSWER: B
What characteristic defines a linear system?
A. Its output is always proportional to its input.
B. It possesses the property of superposition (additivity and scaling).
C. Its behavior changes over time.
D. It only responds to positive inputs.
ANSWER: B
The additivity property for a linear system states that the response to \(x_1(t)+x_2(t)\) is what?
A. \(y_1(t) \cdot y_2(t)\)
B. \(y_1(t)+y_2(t)\)
C. \(y_1(t)-y_2(t)\)
D. \(y_1(t)\) only
ANSWER: B
What is the scaling (or homogeneity) property for a linear system?
A. The response to \(ax_1(t)\) is \(a+y_1(t)\).
B. The response to \(ax_1(t)\) is \(y_1(t)\).
C. The response to \(ax_1(t)\) is \(a y_1(t)\).
D. The response to \(ax_1(t)\) is \(x_1(t)/a\).
ANSWER: C
A direct consequence of the superposition property for linear systems is that an input which is zero for all time results in what?
A. A non-zero output for some time.
B. An output which is zero for all time.
C. An undefined output.
D. An output determined by system memory.
ANSWER: B
A system like \(y[n] = 2x[n] + 3\) is described as nonlinear because it violates which property of linear systems?
A. Additivity property.
B. Homogeneity property.
C. Zero-input/zero-output property.
D. All of the above.
ANSWER: D
