Introduction to Op Amps

Evolution of Signal Conditioning

Historical Context

  • Early systems used vacuum tubes and discrete transistors.
  • High component counts led to reliability and power issues.

The Integrated Circuit (IC) Revolution

  • Shift from discrete to integrated solutions.
  • Reduced size, power, and cost.
  • The Operational Amplifier (Op Amp) emerged as the fundamental building block.

Historically, signal conditioning required designing complex circuits from individual components. Today, ECE engineers use Op Amps as “black box” components. This allows us to focus on the transfer function rather than internal transistor biasing. We prioritize familiarity with specifications and limitations.

Anatomy of the Op Amp

Key Terminals:

  • Inverting (-): Output is \(180^\circ\) out of phase.
  • Non-Inverting (+): Output is in phase.
  • Output: The result of the differential gain.
  • Power Supplies: Usually bipolar (\(\pm V_s\)).

Note

In schematic diagrams, power supply connections are often omitted for clarity, but they are essential for physical operation.

The standard triangle symbol is universal. Note that while we see two inputs and one output, the device is a complex circuit of resistors, transistors, and diodes. The output is always relative to a common ground, even if no ground terminal exists on the op amp itself.

The Transfer Function and Saturation

The output voltage \(V_{out}\) is determined by the differential input: \(V_d = V_1 - V_2\).

\[V_{out} = A(V_1 - V_2)\]

Key Characteristics: - Open-loop Gain (\(A\)): Extremely high (\(>100,000\) V/V). - Linear Range: Very narrow (\(\Delta V < 1\) mV). - Saturation: Output cannot exceed the supply rails (\(\pm V_{sat}\)).

Because the gain is so high, the smallest difference in input drives the output to its maximum or minimum limit. This is why we almost always use “feedback” to control the gain and keep the amplifier in the linear region.

Ideal Op Amp Analysis Rules

To simplify design, we use two fundamental rules assuming an Ideal Op Amp:

Rule 1: No Input Current

Assume the input impedance is infinite. No current flows into the (+) or (-) terminals.

Rule 2: Virtual Short

Assume the gain is infinite. The op amp will adjust its output via feedback to ensure \(V_+ = V_-\).

These rules allow us to use Kirchhoff’s Current Law (KCL) at the input nodes (summing points) to derive transfer functions for complex circuits without knowing the internal transistor physics.

The Inverting Amplifier

Circuit Analysis: Summing currents at the (-) terminal: \(I_1 + I_2 = 0\) \(\frac{V_{in}}{R_1} + \frac{V_{out}}{R_2} = 0\)

Transfer Function: \[V_{out} = -\frac{R_2}{R_1} V_{in}\]

Design Insight: This circuit provides gain and inversion. Input impedance is exactly \(R_1\).

Notice the negative sign; this indicates a \(180^\circ\) phase shift. If \(R_2 < R_1\), the circuit acts as an attenuator. If \(R_2 > R_1\), it is an amplifier.

Non-Ideal Realities

In the real world, we must account for:

  • Finite Open-loop Gain (\(A\)): Typically 200,000.
  • Input Offset Voltage (\(V_{ios}\)): Output might not be zero when input is zero.
  • Slew Rate: Speed limit of the output change (\(V/\mu s\)).
  • Output Current: Most ICs can only sink/source \(\approx 20\) mA.

Engineering Pro-Tip

“Think mA and k\(\Omega\).” Using \(1\Omega\) resistors will exceed the Op Amp’s current capability.

While the ideal rules work 99% of the time, high-precision instrumentation requires compensating for offsets. The book illustrates that a 0.05% error is common when assuming ideal vs real gain.

Voltage Follower (Buffer)

  • Gain: Unity (\(1\) V/V).
  • Input Impedance: Extremely high (\(>100 M\Omega\)).
  • Output Impedance: Very low (\(<100 \Omega\)).

Application: Used as an Impedance Transformer. It prevents a high-impedance sensor from being “loaded” by the next stage of the circuit.

The voltage follower is essentially a non-inverting amplifier with \(R_1 = \infty\) and \(R_2 = 0\). It is the “security guard” of signals, ensuring the sensor voltage stays accurate regardless of the load.

Summing Amplifier

Used to combine multiple signals or add a “bias” (offset) to a signal.

\[V_{out} = -\left[ \frac{R_2}{R_1} V_1 + \frac{R_2}{R_3} V_2 \right]\]

Example Engineering Problem: Level-shifting a sensor output from (0 to 1V) to (5 to 6V). - Input 1: Sensor Signal - Input 2: Fixed DC Bias

By choosing resistor ratios, we can scale each input differently. This is the heart of analog computation.

Non-Inverting Amplifier

Analysis: The voltage at the summing point is \(V_{in}\) (Rule 2). KCL at summing point: \(\frac{V_{in}}{R_1} + \frac{V_{in} - V_{out}}{R_2} = 0\)

Transfer Function: \[V_{out} = \left[ 1 + \frac{R_2}{R_1} \right] V_{in}\]

Advantage: Very high input impedance because the signal goes directly into the Op Amp terminal.

Note that the gain is always \(\ge 1\). This circuit cannot attenuate signals.

Instrumentation Amplifier

The “Gold Standard” for sensor bridge circuits (like Wheatstone bridges).

Key Benefits: 1. High CMRR: Rejects noise common to both wires. 2. High Input Impedance: Dual voltage followers. 3. Single Resistor Gain Control: Gain is set by \(R_G\).

\[V_{out} = \left(1 + \frac{2 R_1}{R_G}\right) \frac{R_3}{R_2} (V_2 - V_1)\]

In industrial environments, wires pick up 60Hz hum. Since the hum is on both wires, the Instrumentation Amp subtracts it out, leaving only the sensor data.

Dynamic Signal Conditioning

Integrator

Used for averaging or ramp generation. \(V_{out} = -\frac{1}{RC} \int V_{in} dt\)

Differentiator

Responds to the rate of change of the input. \(V_{out} = -RC \frac{dV_{in}}{dt}\)

Warning

Differentiators are often unstable in practice because they amplify high-frequency noise.

An integrator with a constant DC input produces a linear ramp. This is the basis for many Analog-to-Digital converters.

Design Guidelines: A Systematic Approach

When designing analog signal conditioning, follow these steps:

  1. Define Objective: What is the parameter, range, and required accuracy?
  2. Sensor Selection: Identify the transfer function (\(R \to T\), \(V \to P\)).
  3. S/C Design:
    • Create a linear equation: \(V_{out} = m \cdot V_{in} + V_0\).
    • Match impedance.
    • Check for loading.
  4. Practical Check: Is the power dissipation safe? Are currents in mA?

Always start with the math. If you can write the equation \(V_{out} = f(V_{in})\), you can build the circuit.

Design Example: Sensor Interface

Problem: Sensor outputs \(20\) to \(250\) mV. We need \(0\) to \(5\) V output.

Step 1: The Equation \(0 = m(0.02) + V_0\) \(5 = m(0.25) + V_0\) Solving gives: \(m = 21.7\), \(V_0 = -0.434\) V.

Step 2: The Implementation \[V_{out} = 21.7 (V_{in} - 0.02)\]

Result: A differential amplifier with gain \(21.7\) and a \(0.02\) V reference on the inverting terminal.

This example shows how we translate a range requirement into a standard Op Amp configuration. Using a Zener diode to provide the 0.02V bias ensures the system remains stable even if the power supply fluctuates.

Summary of Op Amp Configurations

Circuit Function Transfer Equation
Inverting Gain + Inversion \(V_{out} = -(R_2/R_1)V_{in}\)
Non-Inverting High Z Gain \(V_{out} = (1 + R_2/R_1)V_{in}\)
Follower Impedance Buffer \(V_{out} = V_{in}\)
Diff Amp Subtraction \(V_{out} = (R_2/R_1)(V_2 - V_1)\)
Integrator Integration \(V_{out} = -1/RC \int V_{in} dt\)

Each of these circuits serves a specific role in the instrumentation chain. Choosing the right one depends on whether you need to invert, provide high input impedance, or perform mathematical operations.
