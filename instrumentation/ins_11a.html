Welcome students to the first module of Instrumentation. We are transitioning from simple measurement to active control. Our goal today is to define what ‘control’ means in an industrial context and how we represent these systems mathematically and visually.

What is Control?

Control refers to methods used to force parameters in an environment to have specific values.

Natural Control

Biological functions like:

  • Body temperature regulation.
  • Fluid flow in organisms.
  • Glucose levels.

Artificial Control

Engineering solutions like:

  • Room temperature (\(21^{\circ}\mathrm{C}\)).
  • IC Manufacturing.
  • Spacecraft navigation.

Control isn’t just an engineering concept; it’s a fundamental principle of life (Homeostasis). In ECE, we take these logical strategies and implement them using hardware and software.

Process Principles: Self-Regulation

A process is Self-Regulating if the variable adopts a stable value for a given input without external intervention.

  • Example: A tank where \(Q_{\mathrm{out}} = K\sqrt{h}\).
  • As height \(h\) increases, \(Q_{\mathrm{out}}\) increases.
  • Eventually, \(Q_{\mathrm{out}} = Q_{\mathrm{in}}\) and the level stabilizes.

Important

Self-regulation does not provide regulation to a specific setpoint; it simply finds an equilibrium.

Self-regulation is the “natural” state of the system. In the tank example, if you increase the input flow, the level will find a new equilibrium, but it won’t return to the original level unless we add a control loop.

The Evolution of Control

Human-Aided

  • Sensor: Human eye (sight tube).
  • Controller: Human brain (decision).
  • Actuator: Human hand (turning valve).

Automatic

  • Sensor: Electronic transducer.
  • Controller: Microprocessor/PLC.
  • Actuator: Motorized/Pneumatic valve.

The logic remains identical between human-aided and automatic control. We are simply replacing the biological components with faster, more reliable mechanical and electronic counterparts.

Types of Control Systems

Type Objective Example
Process Control Regulate a variable to a constant setpoint. Temperature in a chemical vat.
Servomechanism Force a variable to track a moving reference. Robot arm following a path.
Discrete-State Manage a sequence of start/stop events. Paint mixing schedule (PLCs).

Distinguish these clearly: Process control wants things to stay the same (regulation). Servos want things to follow a specific movement (tracking). Discrete-state is binary (on/off logic).

Key Definitions

  • Controlled Variable (\(c\)): The specific parameter we wish to maintain (e.g., Temperature, Level).
  • Manipulated Variable (\(u\)): The parameter we change to affect the process (e.g., Valve position, Heater voltage).
  • Sensor: Converts the controlled variable into an analog signal (voltage, current).
  • Error Detector: Determines \(e = r - b\).

Sensor vs. Transducer

All sensors are transducers (converting energy), but not all transducers are sensors (e.g., an I/P converter converts current to pressure but doesn’t measure the process).

Evaluation: The 3 Pillars of Performance

How do we know if our control system is “good”?

  1. Stability: The system must not oscillate out of control or diverge.
  2. Steady-State Regulation: The error should be minimized during normal operation (within allowable \(\pm \Delta c\)).
  3. Transient Regulation: How the system handles sudden changes (setpoint shifts or disturbances).

Stability is non-negotiable. Steady-state is about accuracy over time. Transient is about speed and “smoothness” during changes.

Standard Tuning Criteria

When a system is cyclic, we use two primary metrics:

1. Minimum Area

Adjusts tuning so the total integral of the absolute error is minimized: \[A = \int |e(t)| dt\]

2. Quarter Amplitude

Specifies that each successive peak is 1/4 the height of the previous peak: \[a_{n+1} = \frac{a_n}{4}\]

Quarter amplitude is a classic “rule of thumb” in industrial tuning. It provides a good balance between fast response and stability.

Summary

  • Process Control is about maintaining a variable at a setpoint via a feedback loop.
  • The Loop consists of: Process \(\rightarrow\) Sensor \(\rightarrow\) Error Detector \(\rightarrow\) Controller \(\rightarrow\) Actuator.
  • Evaluation is based on how quickly and accurately the system reaches the setpoint without becoming unstable.

Problem Set 1.1

Problem 1: Self-Regulation Calculation

A self-regulating water tank has a flow-to-level relationship defined by: \[Q_{\mathrm{out}} = K\sqrt{h}\]

Given:

  • The constant \(K = 0.85 \text{ (gal/min)}/\text{ft}^{1/2}\)
  • The input flow rate \(Q_{\mathrm{in}}\) is maintained at \(2.5 \text{ gal/min}\).

Task:

Calculate the height \(h\) at which the tank level will stabilize.

This problem tests the student’s understanding of self-regulation equilibrium. At equilibrium, the output flow must exactly match the input flow.

Solution 1: Equilibrium Analysis

To find the stabilized height, we set \(Q_{\mathrm{out}} = Q_{\mathrm{in}}\):

\[2.5 = 0.85\sqrt{h}\]

Step 1: Isolate the square root \[\sqrt{h} = \frac{2.5}{0.85}\]

Step 2: Square both sides \[h = \left(\frac{2.5}{0.85}\right)^2\]

Try it yourself with the solver below:

Explain that if \(Q_{in}\) increases, the level \(h\) must rise to increase the “pressure” (head) driving \(Q_{out}\) until they match again. This is the essence of self-regulation.

Problem 3: Performance Evaluation

A process engineer is tuning a temperature controller using the Quarter-Amplitude Criterion.

The system suffers a transient disturbance. The first overshoot peak (\(a_1\)) is measured at \(8.0^{\circ}\mathrm{C}\) above the setpoint.

Tasks:

  1. Calculate the required amplitude of the second peak (\(a_2\)).
  2. Calculate the required amplitude of the third peak (\(a_3\)).
  3. If the second peak is actually measured at \(3.0^{\circ}\mathrm{C}\), is the system overdamped or underdamped relative to the criterion?

Quarter-amplitude is a specific ratio (4:1) between successive peaks. This problem requires basic geometric sequence calculation and qualitative comparison.

Solution 3: Tuning Criteria

1. Calculate Second Peak (\(a_2\)): \[a_2 = \frac{a_1}{4} = \frac{8.0}{4} = 2.0^{\circ}\mathrm{C}\]

2. Calculate Third Peak (\(a_3\)): \[a_3 = \frac{a_2}{4} = \frac{2.0}{4} = 0.5^{\circ}\mathrm{C}\]

3. Comparison:

  • The target \(a_2\) was \(2.0^{\circ}\mathrm{C}\).
  • The actual \(a_2\) was \(3.0^{\circ}\mathrm{C}\).
  • Since \(3.0 > 2.0\), the oscillations are decaying slower than the quarter-amplitude target. The system is “less damped” than desired.

Remind students that “Minimum Area” is another criterion. In this case, the actual response has a larger area under the curve than the quarter-amplitude target, making it less ideal by both standards.
