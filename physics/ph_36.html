3.6 Finding Velocity and Displacement from Acceleration

Using Integral Calculus

Introduction to Calculus in Kinematics

Previously, we used derivatives to find velocity from position and acceleration from velocity.

Now, we reverse the process using integral calculus to find:

  • Velocity from acceleration.
  • Position from velocity.

This approach is powerful, especially for cases where acceleration is not constant (i.e., acceleration is a function of time, \(a(t)\)).

Note

This section requires familiarity with basic integration concepts. It extends our toolkit for analyzing complex motion.

Kinematic Equations from Integral Calculus

Recall the definitions:

  • Acceleration is the time derivative of velocity: \(\frac{d}{dt}v(t) = a(t)\)
  • Velocity is the time derivative of position: \(\frac{d}{dt}x(t) = v(t)\)

To work backward, we integrate:

Finding Velocity from Acceleration

Given an acceleration function \(a(t)\), we integrate to find velocity \(v(t)\):

\[ \int \frac{d}{dt}v(t) \, dt = \int a(t) \, dt \]

\[ v(t) = \int a(t) \, dt + C_1 \]

Where \(C_1\) is the constant of integration, determined by the initial velocity (\(v_0\)) at time \(t=0\).

If \(v(0) = v_0\), then \(C_1 = v_0\).

The constant of integration \(C_1\) is physically significant: it represents the initial velocity. Without an initial condition, we can only find the general form of the velocity function.

Finding Position from Velocity

Given a velocity function \(v(t)\), we integrate to find position \(x(t)\):

\[ \int \frac{d}{dt}x(t) \, dt = \int v(t) \, dt \]

\[ x(t) = \int v(t) \, dt + C_2 \]

Where \(C_2\) is the constant of integration, determined by the initial position (\(x_0\)) at time \(t=0\).

If \(x(0) = x_0\), then \(C_2 = x_0\).

Similarly, \(C_2\) represents the initial position. These integration constants are crucial for defining the specific motion of a particle.

Deriving Kinematic Equations for Constant Acceleration

Letâ€™s use these integral formulations to re-derive the kinematic equations for constant acceleration, \(a(t) = a\).

Step 1: Find \(v(t)\) from \(a(t) = a\) \[ v(t) = \int a \, dt + C_1 = at + C_1 \] Using initial condition \(v(0) = v_0\): \[ v_0 = a(0) + C_1 \Rightarrow C_1 = v_0 \] So, the velocity equation is: \[ v(t) = v_0 + at \]

(This matches our earlier Equation 3.12!)

This derivation shows how our previously assumed kinematic equations are consistent with calculus. It grounds them in a more fundamental mathematical framework.

Deriving Kinematic Equations (Continued)

Step 2: Find \(x(t)\) from \(v(t) = v_0 + at\) \[ x(t) = \int (v_0 + at) \, dt + C_2 \] \[ x(t) = v_0t + \frac{1}{2}at^2 + C_2 \] Using initial condition \(x(0) = x_0\): \[ x_0 = v_0(0) + \frac{1}{2}a(0)^2 + C_2 \Rightarrow C_2 = x_0 \] So, the position equation is: \[ x(t) = x_0 + v_0t + \frac{1}{2}at^2 \] (This matches our earlier Equation 3.13!)

These derivations highlight that the kinematic equations for constant acceleration are special cases of a more general calculus-based approach. The power of calculus lies in its ability to handle cases where acceleration is not constant.

Example: Motion of a Motorboat

Problem: A motorboat travels at constant \(v = 5.0\) m/s, then starts to accelerate with \(a(t) = -\frac{1}{4}t \, \text{m/s}^3\) to dock.

  1. What is the velocity function \(v(t)\)?

  2. At what time does \(v(t)\) reach zero?

  3. What is the position function \(x(t)\)?

  4. What is the displacement from the start of acceleration until \(v=0\)?

  5. Graph \(v(t)\) and \(x(t)\).

Strategy:

  • Set \(t=0\) when acceleration begins.
    1. Integrate \(a(t)\) to get \(v(t)\), use \(v(0) = 5.0\) m/s for \(C_1\).
    1. Set \(v(t)=0\) and solve for \(t\).
    1. Integrate \(v(t)\) to get \(x(t)\), use \(x(0) = 0\) for \(C_2\).
    1. Evaluate \(x(t)\) at the time found in (b).

This problem demonstrates the step-by-step application of integral calculus to derive motion functions from acceleration. Pay close attention to units and constants of integration.

Example: Solution (Part a)

Solution (Part a): Velocity function \(v(t)\)

Given \(a(t) = -\frac{1}{4}t \, \text{m/s}^3\).

Integrate \(a(t)\): \[ v(t) = \int a(t) \, dt + C_1 = \int (-\frac{1}{4}t) \, dt + C_1 \] \[ v(t) = -\frac{1}{4} \frac{t^2}{2} + C_1 = -\frac{1}{8}t^2 \, \text{m/s}^3 + C_1 \]

Use initial condition \(v(0) = 5.0\) m/s: \[ 5.0 \, \text{m/s} = -\frac{1}{8}(0)^2 + C_1 \Rightarrow C_1 = 5.0 \, \text{m/s} \]

So, the velocity function is: \[ v(t) = 5.0 \, \text{m/s} - \frac{1}{8}t^2 \, \text{m/s}^3 \]

The constant \(C_1\) is the initial velocity. Notice how the units work out: \(\text{m/s}^3 \cdot \text{s}^2 = \text{m/s}\).

Example: Solution (Parts b & c)

Solution (Part b): Time when velocity reaches zero

Set \(v(t) = 0\): \[ 0 = 5.0 \, \text{m/s} - \frac{1}{8}t^2 \, \text{m/s}^3 \] \[ \frac{1}{8}t^2 = 5.0 \] \[ t^2 = 40 \] \[ t = \sqrt{40} \approx 6.32 \, \text{s} \]

(We take the positive root for time).

Solution (Part c): Position function \(x(t)\)

Integrate \(v(t) = 5.0 \, \text{m/s} - \frac{1}{8}t^2 \, \text{m/s}^3\): \[ x(t) = \int (5.0 - \frac{1}{8}t^2) \, dt + C_2 \] \[ x(t) = 5.0t \, \text{m/s} - \frac{1}{8}\frac{t^3}{3} \, \text{m/s}^3 + C_2 \] \[ x(t) = 5.0t \, \text{m/s} - \frac{1}{24}t^3 \, \text{m/s}^3 + C_2 \]

Use initial condition \(x(0) = 0\) (starting position for displacement): \[ 0 = 5.0(0) - \frac{1}{24}(0)^3 + C_2 \Rightarrow C_2 = 0 \]

So, the position function is: \[ x(t) = 5.0t \, \text{m/s} - \frac{1}{24}t^3 \, \text{m/s}^3 \]

The constant \(C_2\) is the initial position. Again, verify units: \(\text{m/s} \cdot \text{s} = \text{m}\) and \(\text{m/s}^3 \cdot \text{s}^3 = \text{m}\).

Example: Solution (Parts d & e)

Solution (Part d): Displacement when velocity is zero

Evaluate \(x(t)\) at \(t \approx 6.32\) s: \[ x(6.32 \, \text{s}) = 5.0(6.32) - \frac{1}{24}(6.32)^3 \] \[ x(6.32 \, \text{s}) \approx 31.6 - \frac{1}{24}(252.8) \] \[ x(6.32 \, \text{s}) \approx 31.6 - 10.53 \approx 21.1 \, \text{m} \]

Solution (Part e): Graphs

Figure 3.30: (a) Velocity of the motorboat. (b) Position of the motorboat.

  • Velocity (a): Starts at 5 m/s, decreases quadratically to 0 at \(t \approx 6.3\) s, then becomes negative.
  • Position (b): Starts at 0, increases to a maximum displacement of \(\approx 21.1\) m at \(t \approx 6.3\) s (where velocity is zero), then decreases as the boat reverses.

The graphs visually confirm the calculations. The maximum position occurs when velocity is zero, which is a general principle. The negative velocity after 6.3s indicates a change in direction, consistent with the boat stopping and then potentially reversing if the acceleration continues.

Check Your Understanding 3.8

Problem: A particle starts from rest and has an acceleration function \(a(t) = (5 - (10/1 \, \text{s})t) \, \text{m/s}^2\).

  1. What is the velocity function?

  2. What is the position function?

  3. When is the velocity zero?

Given:

  • \(a(t) = (5 - 10t) \, \text{m/s}^2\) (assuming \(1/s\) is unit)
  • Starts from rest: \(v(0) = 0\)
  • Assume \(x(0) = 0\) for position function.

Steps:

  1. Integrate \(a(t)\) to find \(v(t)\).
  2. Use \(v(0)=0\) to find \(C_1\).
  3. Integrate \(v(t)\) to find \(x(t)\).
  4. Use \(x(0)=0\) to find \(C_2\).
  5. Set \(v(t)=0\) and solve for \(t\).

Guide students through the steps:

  1. \(v(t) = \int (5 - 10t) dt = 5t - 5t^2 + C_1\). Since \(v(0)=0\), \(C_1=0\). So \(v(t) = 5t - 5t^2 \, \text{m/s}\).

  2. \(x(t) = \int (5t - 5t^2) dt = \frac{5}{2}t^2 - \frac{5}{3}t^3 + C_2\). Since \(x(0)=0\), \(C_2=0\). So \(x(t) = \frac{5}{2}t^2 - \frac{5}{3}t^3 \, \text{m}\).

  3. Set \(v(t) = 5t - 5t^2 = 0 \Rightarrow 5t(1 - t) = 0\). Solutions are \(t=0\) s and \(t=1\) s. The velocity is zero at \(t=1\) s (after starting).

Key Takeaways

  • Calculus Link: Integration is the inverse of differentiation in kinematics.
    • \(\int a(t) \, dt = v(t) + C_1\)
    • \(\int v(t) \, dt = x(t) + C_2\)
  • Constants of Integration: \(C_1\) is the initial velocity (\(v_0\)), and \(C_2\) is the initial position (\(x_0\)).
  • General Motion: This method allows for analyzing motion where acceleration is a known function of time, not just constant.
  • Derivation Power: Integral calculus provides a rigorous way to derive the kinematic equations for constant acceleration.
  • Problem-Solving:
    • Integrate \(a(t)\) to get \(v(t)\), using \(v_0\).
    • Integrate \(v(t)\) to get \(x(t)\), using \(x_0\).
    • Solve for specific times or positions by setting the functions to zero or specific values.

Key Equations

Equation Description
\(v(t) = \int a(t) \, dt + C_1\) General velocity from acceleration (with initial velocity \(C_1 = v_0\))
\(x(t) = \int v(t) \, dt + C_2\) General position from velocity (with initial position \(C_2 = x_0\))

Key Terms

Term Definition
Integral Calculus A branch of calculus used to find the accumulation of quantities and the areas under curves; applied here to find velocity from acceleration and position from velocity.
Constant of Integration (\(C_1, C_2\)) Arbitrary constants that arise from indefinite integration; determined by initial conditions in physics problems (e.g., initial velocity and initial position).
Functional Form An expression or equation that defines a quantity (like velocity or position) as a function of another variable (like time).
