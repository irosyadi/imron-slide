4.1 Displacement, Velocity vector

Learning Objectives

By the end of this section, you will be able to:

  • Calculate position vectors in a multidimensional displacement problem.
  • Solve for the displacement in two or three dimensions.
  • Calculate the velocity vector given the position vector as a function of time.
  • Calculate the average velocity in multiple dimensions.

These are the key skills and concepts we will cover today. We’ll start with position and displacement, then move on to velocity.

Introduction to Multidimensional Motion

  • Motion in 2D and 3D extends 1D definitions.
  • Requires vector algebra, not scalar algebra.
  • Essential for describing complex trajectories.

In one dimension, we could describe motion using just numbers, positive or negative. Now, with multiple dimensions, direction becomes crucial, so we must use vectors. This allows us to describe more realistic motions like a ball thrown through the air or a satellite orbiting Earth.

Coordinate Systems and Position Vectors

  • To describe motion, first establish a coordinate system (x, y, z).
  • Particle location: \(P(x, y, z)\).
  • Coordinates are functions of time: \(x = x(t)\) \(y = y(t)\) \(z = z(t)\)

Just like in 1D, where we picked an origin, in 2D or 3D, we need a coordinate system. We typically use Cartesian coordinates \((x, y, z)\). As the particle moves, its coordinates change over time.

The Position Vector

  • Vector from origin to point \(P(x, y, z)\).
  • Represented as \(\vec{r}(t)\).
  • In unit vector notation: \[\vec{r}(t) = x(t)\hat{i} + y(t)\hat{j} + z(t)\hat{k}\]

Figure 4.2: A 3D coordinate system with position vector \(\vec{r}(t)\)

Note

Right-Handed Coordinate System

This orientation (x out, y right, z up) is used throughout the chapter.

The position vector \(\vec{r}(t)\) points from the origin to the particle’s location at any given time \(t\). We use unit vectors \(\hat{i}\), \(\hat{j}\), and \(\hat{k}\) to denote the x, y, and z directions, respectively. The image shows a typical right-handed coordinate system. If you curl the fingers of your right hand from the positive x-axis to the positive y-axis, your thumb will point in the direction of the positive z-axis.

Displacement Vector in 3D

  • Change in position of a particle.
  • From initial position \(\vec{r}(t_1)\) to final position \(\vec{r}(t_2)\).
  • Calculated as: \[\Delta\vec{r} = \vec{r}(t_2) - \vec{r}(t_1)\]

Figure 4.3: Displacement vector \(\Delta\vec{r}\) from \(P_1\) to \(P_2\)

Important

Displacement is a vector quantity! It represents the straight-line distance and direction between two points, regardless of the path taken.

Displacement is not the same as total distance traveled. It’s simply the difference between the final and initial position vectors. Think of it as “as the crow flies.” This is a vector subtraction, so we subtract the corresponding components.

Example: Polar Orbiting Satellite (1/2)

A satellite orbits Earth at an altitude of 400 km. What is the displacement vector from the North Pole to \(-45^\circ\) latitude? (Radius of Earth = 6370 km)

Strategy:

  1. Visualize the problem graphically.
  2. Use unit vectors to calculate displacement.

This example helps us apply the displacement vector concept. We’ll set up a coordinate system and define the initial and final positions using unit vectors, then perform the vector subtraction.

Example: Polar Orbiting Satellite (2/2)

Solution:

  • Orbit radius \(R = 6370 \text{ km} + 400 \text{ km} = 6770 \text{ km}\).
  • North Pole is at \(90^\circ\) latitude (along positive y-axis).
  • Initial position vector: \(\vec{r}(t_1) = 6770 \hat{j} \text{ km}\)
  • Final position at \(-45^\circ\) latitude (below x-axis): \(\vec{r}(t_2) = 6770 (\cos(-45^\circ))\hat{i} + 6770 (\sin(-45^\circ))\hat{j} \text{ km}\) \(\vec{r}(t_2) = 4787\hat{i} - 4787\hat{j} \text{ km}\)
  • Displacement vector: \(\Delta\vec{r} = \vec{r}(t_2) - \vec{r}(t_1)\) \(\Delta\vec{r} = (4787\hat{i} - 4787\hat{j}) - (6770\hat{j})\) \(\Delta\vec{r} = 4787\hat{i} - 11,557\hat{j} \text{ km}\)
  • Magnitude: \(|\Delta\vec{r}| = \sqrt{(4787)^2 + (-11,557)^2} = 12,509 \text{ km}\)
  • Direction: \(\theta = \tan^{-1}\left(\frac{-11,557}{4787}\right) = -67.5^\circ\) (south of east)

Figure 4.4: Satellite orbit visualization

Figure 4.5: Displacement vector components

First, we define our coordinate system with the North Pole along the y-axis. The total radius for the satellite’s orbit is the Earth’s radius plus the altitude. Then we find the components of the initial and final position vectors. Remember that negative angles are measured clockwise from the positive x-axis. Finally, we subtract the vectors component-wise to get the displacement, then calculate its magnitude and direction. Note that the path taken by the satellite is curved, but the displacement is a straight line.

Example: Brownian Motion

A particle undergoes Brownian motion with the following displacements (in \(\mu m\)):

\(\Delta\vec{r}_1 = 2.0\hat{i} + \hat{j} + 3.0\hat{k}\)

\(\Delta\vec{r}_2 = -\hat{i} + 3.0\hat{k}\)

\(\Delta\vec{r}_3 = 4.0\hat{i} - 2.0\hat{j} + \hat{k}\)

\(\Delta\vec{r}_4 = -3.0\hat{i} + \hat{j} + 2.0\hat{k}\)

What is the total displacement from the origin?

Solution:

Sum the individual displacement vectors:

\(\Delta\vec{r}_{Total} = \Delta\vec{r}_1 + \Delta\vec{r}_2 + \Delta\vec{r}_3 + \Delta\vec{r}_4\)

\(\Delta\vec{r}_{Total} = (2.0 - 1.0 + 4.0 - 3.0)\hat{i} + (1.0 + 0 - 2.0 + 1.0)\hat{j} + (3.0 + 3.0 + 1.0 + 2.0)\hat{k}\)

\(\Delta\vec{r}_{Total} = 2.0\hat{i} + 0\hat{j} + 9.0\hat{k} \text{ }\mu m\)

Magnitude:

\(|\Delta\vec{r}_{Total}| = \sqrt{(2.0)^2 + (0)^2 + (9.0)^2} = \sqrt{4 + 81} = \sqrt{85} \approx 9.2 \text{ }\mu m\)

Direction (with respect to x-axis in xz-plane): \(\theta = \tan^{-1}\left(\frac{9}{2}\right) \approx 77^\circ\)

Figure 4.6: Trajectory of Brownian motion

This example shows how to sum multiple displacement vectors. We simply add the corresponding components (x, y, and z) separately. Notice that the total displacement is much shorter than the sum of the magnitudes of individual displacements, illustrating the difference between displacement and total path length. The final direction is given with respect to the x-axis in the xz-plane, since the y-component is zero.

Instantaneous Velocity Vector

  • Calculated as the derivative of the position vector with respect to time. \[\vec{v}(t) = \lim_{\Delta t \to 0} \frac{\vec{r}(t+\Delta t) - \vec{r}(t)}{\Delta t} = \frac{d\vec{r}}{dt}\]
  • The velocity vector is always tangent to the particle’s path.

Figure 4.7: Velocity vector tangent to the path

In component form:

\(\vec{v}(t) = v_x(t)\hat{i} + v_y(t)\hat{j} + v_z(t)\hat{k}\)

Where:

\(v_x(t) = \frac{dx(t)}{dt}\)

\(v_y(t) = \frac{dy(t)}{dt}\)

\(v_z(t) = \frac{dz(t)}{dt}\)

Just like in 1D, instantaneous velocity is the rate of change of position. But now, it’s a vector derivative. Graphically, this means the velocity vector at any point on the path will point in the direction the particle is moving at that exact instant. We calculate each component of velocity by taking the derivative of the corresponding position component.

Average Velocity Vector

  • Vector equivalent of 1D average velocity.
  • Calculated over a time interval from \(t_1\) to \(t_2\): \[\vec{v}_{avg} = \frac{\vec{r}(t_2) - \vec{r}(t_1)}{t_2 - t_1}\]
  • This is simply the total displacement divided by the total time taken.

Average velocity is straightforward: find the total displacement (final position minus initial position) and divide it by the total time elapsed. It gives you the average rate and direction of change in position over the interval.

Example: Calculating the Velocity Vector (1/2)

Position function of a particle: \(\vec{r}(t) = 2.0t^2\hat{i} + (2.0 + 3.0t)\hat{j} + 5.0t\hat{k} \text{ m}\)

  1. What is the instantaneous velocity and speed at \(t = 2.0 \text{ s}\)?

  2. What is the average velocity between \(1.0 \text{ s}\) and \(3.0 \text{ s}\)?

Let’s break this down into two parts: instantaneous velocity and average velocity. For instantaneous, we’ll need to take a derivative. For average, we’ll use the displacement formula.

Example: Calculating the Velocity Vector (2/2)

Solution:

  1. Instantaneous velocity and speed at \(t = 2.0 \text{ s}\)

  • Take the derivative of \(\vec{r}(t)\) with respect to time to get \(\vec{v}(t)\):

    \(\vec{v}(t) = \frac{d\vec{r}}{dt} = \frac{d}{dt}(2.0t^2)\hat{i} + \frac{d}{dt}(2.0 + 3.0t)\hat{j} + \frac{d}{dt}(5.0t)\hat{k}\)

    \(\vec{v}(t) = 4.0t\hat{i} + 3.0\hat{j} + 5.0\hat{k} \text{ m/s}\)

  • Substitute \(t = 2.0 \text{ s}\):

    \(\vec{v}(2.0 \text{ s}) = 4.0(2.0)\hat{i} + 3.0\hat{j} + 5.0\hat{k}\)

    \(\vec{v}(2.0 \text{ s}) = 8.0\hat{i} + 3.0\hat{j} + 5.0\hat{k} \text{ m/s}\)

  • Speed is the magnitude of velocity:

    \(|\vec{v}(2.0 \text{ s})| = \sqrt{(8.0)^2 + (3.0)^2 + (5.0)^2} = \sqrt{64 + 9 + 25} = \sqrt{98} \approx 9.9 \text{ m/s}\)

  1. Average velocity between \(1.0 \text{ s}\) and \(3.0 \text{ s}\)

  • Calculate \(\vec{r}(1.0 \text{ s})\) and \(\vec{r}(3.0 \text{ s})\): \(\vec{r}(1.0 \text{ s}) = 2.0(1.0)^2\hat{i} + (2.0 + 3.0(1.0))\hat{j} + 5.0(1.0)\hat{k} = 2.0\hat{i} + 5.0\hat{j} + 5.0\hat{k} \text{ m}\)

    \(\vec{r}(3.0 \text{ s}) = 2.0(3.0)^2\hat{i} + (2.0 + 3.0(3.0))\hat{j} + 5.0(3.0)\hat{k} = 18.0\hat{i} + 11.0\hat{j} + 15.0\hat{k} \text{ m}\)

  • Calculate average velocity:

    \(\vec{v}_{avg} = \frac{\vec{r}(3.0 \text{ s}) - \vec{r}(1.0 \text{ s})}{3.0 \text{ s} - 1.0 \text{ s}}\)

    \(\vec{v}_{avg} = \frac{(18.0\hat{i} + 11.0\hat{j} + 15.0\hat{k}) - (2.0\hat{i} + 5.0\hat{j} + 5.0\hat{k})}{2.0 \text{ s}}\)

    \(\vec{v}_{avg} = \frac{16.0\hat{i} + 6.0\hat{j} + 10.0\hat{k}}{2.0 \text{ s}} = 8.0\hat{i} + 3.0\hat{j} + 5.0\hat{k} \text{ m/s}\)

Note

In this specific case, the average velocity equals the instantaneous velocity at \(t = 2.0 \text{ s}\) because the velocity function is linear. This is generally not true.

For part (a), we differentiate each component of the position vector with respect to time to get the velocity vector. Then, we plug in \(t = 2.0 \text{ s}\). The speed is simply the magnitude of this velocity vector. For part (b), we first find the position vectors at \(t=1.0 \text{ s}\) and \(t=3.0 \text{ s}\). Then we apply the average velocity formula, which is the change in position (displacement) divided by the change in time. It’s interesting to note that in this particular problem, the average velocity matches the instantaneous velocity at the midpoint of the time interval. This happens because the acceleration is constant, meaning the velocity changes linearly with time.

Independence of Perpendicular Motions

  • Motion along one coordinate axis (e.g., x) does not affect motion along perpendicular axes (y, z).
  • Components of position and velocity equations are separate functions of time.
  • This allows us to analyze complex 2D/3D motion by breaking it down into independent 1D motions.

This is a fundamental concept in kinematics. It means we can solve for the x-motion, y-motion, and z-motion independently using our 1D kinematic equations, and then combine the results as vectors.

Illustrating Independence of Motion

Example:

  • A woman walks from A to B in a city with square blocks.
  • Her eastward movement is independent of her northward movement.
  • Total displacement is the vector sum.

Stroboscopic Example:

  • One ball dropped from rest.
  • Another thrown horizontally from the same height.
  • Vertical positions are identical at all times.
  • Horizontal motion is constant for the thrown ball.

Figure 4.8: Two balls illustrating independent motion

The city block example is intuitive: moving east doesn’t stop you from moving north. The stroboscopic photo is a classic demonstration. Despite the horizontal motion of one ball, both balls fall vertically at the same rate. This visually proves that horizontal and vertical motions are independent, assuming negligible air resistance. This principle is crucial for understanding projectile motion.

Key Takeaways

  • Position Vector \(\vec{r}(t)\): Describes a particle’s location from the origin in 2D/3D space, changing with time.
  • Displacement Vector \(\Delta\vec{r}\): The change in position, \(\vec{r}(t_2) - \vec{r}(t_1)\), representing the shortest path between two points.
  • Instantaneous Velocity Vector \(\vec{v}(t)\): The derivative of the position vector with respect to time, \(\frac{d\vec{r}}{dt}\), always tangent to the path.
  • Average Velocity Vector \(\vec{v}_{avg}\): Total displacement divided by the time interval, \(\frac{\Delta\vec{r}}{\Delta t}\).
  • Independence of Motion: Perpendicular components of motion can be analyzed separately, simplifying complex problems (e.g., projectile motion).

These are the most important points to remember from today’s discussion. Understanding these concepts is fundamental to studying more complex motion in physics.

Key Equations

Equation Description
\(\vec{r}(t) = x(t)\hat{i} + y(t)\hat{j} + z(t)\hat{k}\) Position vector in unit vector notation.
\(\Delta\vec{r} = \vec{r}(t_2) - \vec{r}(t_1)\) Displacement vector.
\(\vec{v}(t) = \frac{d\vec{r}}{dt}\) Instantaneous velocity vector.
\(\vec{v}(t) = v_x(t)\hat{i} + v_y(t)\hat{j} + v_z(t)\hat{k}\) Instantaneous velocity components.
\(v_x(t) = \frac{dx(t)}{dt}, v_y(t) = \frac{dy(t)}{dt}, v_z(t) = \frac{dz(t)}{dt}\) Components of instantaneous velocity.
\(\vec{v}_{avg} = \frac{\vec{r}(t_2) - \vec{r}(t_1)}{t_2 - t_1}\) Average velocity vector.

These equations are the mathematical tools we use to describe and analyze displacement and velocity in two and three dimensions. Make sure you understand what each symbol represents and when to use each equation.

Key Terms

Term Definition
Position Vector A vector that points from the origin of a coordinate system to the location of a particle at a specific time.
Displacement Vector The vector representing the change in position of a particle, from an initial point to a final point, independent of the path taken.
Instantaneous Velocity The velocity of a particle at a specific instant in time, given by the derivative of the position vector with respect to time.
Average Velocity The total displacement of a particle divided by the total time taken for that displacement.
Independence of Motion The principle that motion along perpendicular axes of a coordinate system can be analyzed separately.

Understanding these terms is crucial for a strong foundation in kinematics. They form the vocabulary we use to describe motion.
