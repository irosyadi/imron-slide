What is the general term for a rectangular array of numbers?
A. Scalar
B. Vector
C. Matrix
D. Equation
ANSWER: C
In matrix notation, what does \(a_{ij}\) or \((A)_{ij}\) represent?
A. The total number of entries in matrix A.
B. The entry in row \(i\) and column \(j\) of matrix A.
C. The sum of all entries in row \(i\) and column \(j\).
D. The scalar multiplied by matrix A.
ANSWER: B
A matrix with only one row is called a:
A. Column vector
B. Square matrix
C. Row vector
D. Identity matrix
ANSWER: C
For two matrices A and B to be defined as equal, what two conditions must they satisfy?
A. They must have the same number of non-zero entries and the same sum of entries.
B. They must have been generated from the same linear system.
C. They must have the same size and their corresponding entries must be equal.
D. They must commute under multiplication.
ANSWER: C
If A and B are matrices of the same size, how is their sum \(A + B\) computed?
A. By multiplying corresponding entries and summing them.
B. By adding the entries of B to the corresponding entries of A.
C. By adding the first row of A to the first row of B and so on.
D. By cross-multiplying the entries.
ANSWER: B
If \(A\) is any matrix and \(c\) is any scalar, what is the product \(cA\)?
A. A matrix obtained by adding \(c\) to each entry of \(A\).
B. A matrix obtained by multiplying only the first row of \(A\) by \(c\).
C. A matrix obtained by multiplying each entry of \(A\) by \(c\).
D. A matrix obtained by taking the absolute value of each entry of \(A\).
ANSWER: C
For the matrix product \(AB\) to be defined, what condition must be met between the sizes of matrix A and matrix B?
A. The number of rows of A must be the same as the number of columns of B.
B. The number of columns of A must be the same as the number of rows of B.
C. The number of rows of A must be the same as the number of rows of B.
D. The number of columns of A must be the same as the number of columns of B.
ANSWER: B
If A is an \(m \times r\) matrix and B is an \(r \times n\) matrix, what is the size of their product \(AB\)?
A. \(r \times m\)
B. \(n \times m\)
C. \(m \times n\)
D. \(r \times r\)
ANSWER: C
The entry in row \(i\) and column \(j\) of a matrix product \(AB\) is calculated by multiplying corresponding entries from which parts of A and B, and then summing the products?
A. Row \(j\) of A and column \(i\) of B.
B. Column \(i\) of A and row \(j\) of B.
C. Row \(i\) of A and column \(j\) of B.
D. Column \(j\) of A and row \(i\) of B.
ANSWER: C
An expression of the form \(c_1A_1 + c_2A_2 + \dots + c_rA_r\), where \(A_1, \dots, A_r\) are matrices of the same size and \(c_1, \dots, c_r\) are scalars, is called a:
A. Matrix product
B. Vector sum
C. Linear combination
D. Scalar projection
ANSWER: C
If \(A\) is an \(m \times n\) matrix and \(\mathbf{x}\) is an \(n \times 1\) column vector, the product \(A\mathbf{x}\) can be expressed as a linear combination of what?
A. The row vectors of A.
B. The column vectors of \(\mathbf{x}\).
C. The column vectors of A.
D. The row vectors of \(\mathbf{x}\).
ANSWER: C
A system of \(m\) linear equations in \(n\) unknowns can be represented by a single matrix equation in the form:
A. \(A+B=C\)
B. \(A\mathbf{x} = \mathbf{b}\)
C. \(\mathbf{x} = A\mathbf{b}\)
D. \(A^T = B\)

What operation is applied to an \(m \times n\) matrix \(A\) to obtain its transpose \(A^T\)?
A. Multiplying all entries by -1.
B. Interchanging the rows and columns.
C. Reversing the order of its rows.
D. Squaring each entry in the matrix.

C. \(m \times m\)

B. The product of the entries on the main diagonal.
